### 1.3 数据库表设计

#### 数据库名：checkin_db

#### 表 1：签到记录表 check_in_record

```sql
CREATE TABLE check_in_record (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    student_id      VARCHAR(20)  NOT NULL COMMENT '学号',
    student_name    VARCHAR(50)  NOT NULL COMMENT '姓名',
    room_number     VARCHAR(20)  NOT NULL COMMENT '寝室号',
    dorm_building   VARCHAR(50)  NOT NULL COMMENT '宿舍楼',
    status          VARCHAR(20)  DEFAULT 'normal' COMMENT '签到状态(normal/late/absent)',
    location_info   VARCHAR(255) COMMENT 'GPS定位信息',
    check_time      DATETIME     NOT NULL COMMENT '签到时间',
    created_at      DATETIME     DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at      DATETIME     DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
    deleted         TINYINT      DEFAULT 0 COMMENT '逻辑删除(0未删/1已删)',
    INDEX idx_student_id (student_id),
    INDEX idx_check_time (check_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='签到记录表';
```

#### 表 2：请假申请表 leave_request


## 二、功能模块实现：签到打卡

### 2.1 功能设计

- **输入**：学生通过 AI 对话输入学号、寝室号、宿舍楼
- **处理**：系统采集 GPS 定位，验证位置是否在签到范围内（半径 25m）
- **输出**：生成签到记录，返回签到结果

### 2.2 技术架构

```
前端(Thymeleaf) → Controller → Service → Mapper → MySQL
                  ↑ AI对话(LangChain4j) ↑
                  意图识别 → 多轮对话采集信息
```

**Service: CheckInService.java**

```java
@Service
public class CheckInService {

    @Autowired
    private CheckInRecordMapper checkInRecordMapper;

    public Result processCheckIn(CheckInDTO dto) {
        // 1. 验证位置信息
        if (!validateLocation(dto.getLocationInfo())) {
            return Result.fail("不在签到范围内");
        }
        // 2. 构建签到记录
        CheckInRecord record = new CheckInRecord();
        BeanUtils.copyProperties(dto, record);
        record.setStatus("normal");
        record.setCheckTime(LocalDateTime.now());
        // 3. 保存记录
        checkInRecordMapper.insert(record);
        return Result.success("签到成功");
    }

    private boolean validateLocation(String locationInfo) {
        // 验证 GPS 坐标是否在签到范围(25m)内

    }
}
```
