# 校园寝室自助签到系统 - 数据库设计文档

## 一、数据库概述

- **数据库名称**：`checkin_db`
- **数据库类型**：MySQL 8.x
- **字符集**：utf8mb4_unicode_ci
- **运行环境**：WSL Docker 容器 `1Panel-mysql-RIP6`

---

## 二、数据表结构

### 1. chat_history（聊天历史记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键ID |
| session_id | VARCHAR(255) | INDEX | 会话ID |
| student_id | VARCHAR(50) | INDEX | 学生学号 |
| role | VARCHAR(50) | | 角色(user/assistant/system) |
| content | LONGTEXT | | 消息内容 |
| created_date | DATETIME | DEFAULT NOW() | 创建时间 |
| updated_date | DATETIME | ON UPDATE NOW() | 更新时间 |

**用途**：AOP 拦截后保存的对话溯源记录，支持辅导员审计。

### 2. check_in_record（寝室签到记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键ID |
| student_id | VARCHAR(50) | NOT NULL, INDEX | 学生学号 |
| student_name | VARCHAR(100) | | 学生姓名 |
| room_number | VARCHAR(20) | | 寝室号 |
| dorm_building | VARCHAR(50) | | 宿舍楼栋 |
| status | VARCHAR(20) | DEFAULT '已签到' | 签到状态 |
| location_info | VARCHAR(255) | | 位置信息 |
| check_time | DATETIME | INDEX | 签到时间 |
| created_date | DATETIME | DEFAULT NOW() | 创建时间 |
| updated_date | DATETIME | ON UPDATE NOW() | 更新时间 |

**用途**：保存学生每日的寝室签到记录。

### 3. leave_request（请假申请表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键ID |
| student_id | VARCHAR(50) | NOT NULL, INDEX | 学生学号 |
| student_name | VARCHAR(100) | | 学生姓名 |
| leave_type | VARCHAR(20) | | 请假类型(病假/事假/回家/其他) |
| start_time | DATETIME | | 开始时间 |
| end_time | DATETIME | | 结束时间 |
| reason | TEXT | | 请假原因 |
| audit_status | VARCHAR(20) | DEFAULT '待审核', INDEX | 审核状态 |
| auditor_comment | TEXT | | 审核意见 |
| created_date | DATETIME | DEFAULT NOW() | 创建时间 |
| updated_date | DATETIME | ON UPDATE NOW() | 更新时间 |

**用途**：保存学生的请假申请及审核状态。

## 四、索引策略

| 表名 | 索引字段 | 索引类型 | 用途 |
|------|---------|---------|------|
| chat_history | session_id | 普通索引 | 快速查询会话历史 |
| chat_history | student_id | 普通索引 | 按学号查询记录 |
| check_in_record | student_id | 普通索引 | 按学号查询签到 |
| check_in_record | check_time | 普通索引 | 按时间范围查询 |
| leave_request | student_id | 普通索引 | 按学号查询请假 |
| leave_request | audit_status | 普通索引 | 按审核状态筛选 |

## 五、建表 SQL

```sql
CREATE DATABASE IF NOT EXISTS checkin_db DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
USE checkin_db;

CREATE TABLE chat_history (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    created_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    session_id VARCHAR(255) DEFAULT NULL COMMENT '会话ID',
    student_id VARCHAR(50) DEFAULT NULL COMMENT '学生学号',
    role VARCHAR(50) DEFAULT NULL COMMENT '角色',
    content LONGTEXT COMMENT '消息内容',
    INDEX idx_session_id (session_id),
    INDEX idx_student_id (student_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='聊天历史记录';

CREATE TABLE check_in_record (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    created_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    student_id VARCHAR(50) NOT NULL COMMENT '学生学号',
    student_name VARCHAR(100) DEFAULT NULL COMMENT '学生姓名',
    room_number VARCHAR(20) DEFAULT NULL COMMENT '寝室号',
    dorm_building VARCHAR(50) DEFAULT NULL COMMENT '宿舍楼栋',
    status VARCHAR(20) DEFAULT '已签到' COMMENT '签到状态',
    location_info VARCHAR(255) DEFAULT NULL COMMENT '位置信息',
    check_time DATETIME DEFAULT NULL COMMENT '签到时间',
    INDEX idx_student_id (student_id),
    INDEX idx_check_time (check_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='寝室签到记录';

CREATE TABLE leave_request (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    created_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_date DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    student_id VARCHAR(50) NOT NULL COMMENT '学生学号',
    student_name VARCHAR(100) DEFAULT NULL COMMENT '学生姓名',
    leave_type VARCHAR(20) DEFAULT NULL COMMENT '请假类型',
    start_time DATETIME DEFAULT NULL COMMENT '开始时间',
    end_time DATETIME DEFAULT NULL COMMENT '结束时间',
    reason TEXT COMMENT '请假原因',
    audit_status VARCHAR(20) DEFAULT '待审核' COMMENT '审核状态',
    auditor_comment TEXT COMMENT '审核意见',
    INDEX idx_student_id (student_id),
    INDEX idx_audit_status (audit_status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='请假申请';
```
